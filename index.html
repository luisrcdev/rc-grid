Las Filas

Las filas son los contenedores principales (padres), cuentan con un ancho definido en cada tamaño de pantalla para aplicar el diseño responsivo de manera adecuada

Nota: No pueden existir filas sin cajas adentro. Tampoco cajas sin una fila como contenedor.

Las Cajas

Las Cajas son las columnas que se utilizan para dividir el contenido dentro de nuestra página. Puedes agregar las que necesites dentro de una fila, por defecto tienen un ancho fijo responsivo.
